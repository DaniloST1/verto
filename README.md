# Verto - Plataforma de Gestão de Licitações 🚀

Verto é uma plataforma moderna e responsiva voltada para a gestão de editais, contratos e fluxo de caixa de licitações. Desenvolvida com as tecnologias mais atuais para oferecer uma experiência premium e eficiente.

## 🛠️ Tecnologias Utilizadas

- **React 19** - UI moderna e rápida.
- **Vite** - Build ultrarrápido.
- **Supabase** - Banco de Dados, Autenticação e Storage.
- **Lucide React** - Ícones profissionais.
- **Recharts** - Dashboards financeiros e gráficos dinâmicos.
- **CSS Vanilla** - Design customizado com estética glassmorphism.

## ✨ Funcionalidades

- **Dashboard Financeiro:** Visão clara de receitas, despesas e distribuição de editais.
- **Gestão de Editais:** Cadastro completo com vinculação de clientes.
- **Controle de Pagamentos:** Gestão de mensalidades com status automatizado (Pago, Pendente, Atrasado).
- **Gestão de Usuários:** Controle de perfis com upload de avatares.
- **Responsividade Total:** Adaptado para Computadores, Tablets e Mobile.

## 🚀 Como rodar localmente

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Crie um arquivo `.env` com suas credenciais do Supabase:
   ```env
   VITE_SUPABASE_URL=seu_url
   VITE_SUPABASE_ANON_KEY=seu_key
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
